# Workspace Mandates

## Tech Stack & Architecture
- **Framework:** Always use **React (Vite)** for frontend development.
- **Styling:** Always use **Tailwind CSS**.
- **Component Style:** Use a clean **Functional Component** style with hooks.

## Development Workflow
- **Git Strategy:** Always use a **multi-branch Git strategy** for every feature. Create a new branch for each feature and merge it back to the main branch upon completion.
